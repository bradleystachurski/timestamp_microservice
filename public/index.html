API: Timestamp Microservice

This is a simple microservice that takes string parameters in the URL. Anything following '/' will be processed.

The string passed in through the URL can be formatted either as a unix timestamp or a natural language date.

See the examples below and give it a try.

Example usage:

https://timestamp-stachurski.herokuapp.com/October%201,%202016
https://timestamp-stachurski.herokuapp.com/1475269200

Example output:

{ "unix": 1475269200, "natural": "October 1, 2016" }
